# vim-headtail

vim-headtail is a Vim plugin to move cursor to head or tail of textobj.

## Installation

```vim
call dein#add('utubo/vim-headtail')
HeadTailMap [ ]
```

You can type `[it` to jump to the head of inner of html-tag.
